## Installation
1) [Install](https://arduino.github.io/arduino-cli/installation/) the Arduino CLI tools
2) Download the folder and install
```
$ cd /path/to/configurator
$ npm install -g
```

## How to use
### Create a new file
```
$ cd /path/to/configurator
$ configurator
```

### Edit an existing session file
```
$ configurator -s /path/to/file.txt
```
